Code voorbeelden

Lazy Loading

Loading is een attribuut waarmee wij ervoor kunnen zorgen dat bepaalde foto's die bij loading="lazy" hebben, alleen worden ingeladen wanneer zij te zien zijn voor de gebruiker. Hiermme kunnen we ervoor zorgen dat niet alles van de pagina meteen wordt ingeladen waardoor de pagespeed en de algemene performance van de website wordt verbetert.

Een voorbeeld van Lazy Loading

<img id="uitleg-container-wide-img" src="/assets/wettelijk.png" loading="lazy" alt="Foto over de wettelijke achtergrond">

Responsive images

Responsive images is een manier waarmee je verschillende formaten of versies van een bepaalde foto kan inladen afhankelijk van de viewport width. Door middel van responsive images hebben wij ervoor gezorgd dat op kleinere schermen een cropped image op de plaats komt van de originele image aangezien de originele foto toch niet helemaal te zien is op mobiele versies. Door deze toepassingen wordt op mobiel de kleinere fotos ingeladen waardoor de performance daarop wordt verbetert.
